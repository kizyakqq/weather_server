# ***УСТАНОВКА И ЗАПУСК***

1. Клонируйте репозиторий

        mkdir weather_server && cd weather_server

        git clone https://github.com/kizyakqq/weather_server.git

2. Создайте виртуальное окружение

        python -m venv venv

3. Активируйте виртуальное окружение

        source venv\Scripts\activate

4. Установите зависимости

        pip install -r requirements.txt

5. Запустите сервер

        python script.py

# ***API Endpoints***

- POST /weather/current (Получение текущей погоды по координатам)
- POST /cities (Добавление города для отслеживания)
- GET /cities (Получение списка отслеживаемых городов)
- GET /weather/{city_name}/{time} (Получение погоды для города на указанное время):

    Возвращает погодные данные для указанного города и времени. Сервер ищет в БД ближайшую запись к запрошенному времени в пределах текущего дня.
  
    *Параметры пути:*
  
        city_name — Название города (например, Moscow)
  
        time — Время в формате ISO 8601 (например, 2026-02-16T12:00:00)
  
    *Параметры запроса (опционально):*

  params — Список запрашиваемых параметров. Возможные значения:
  
            temperature
            humidity
            wind_speed
            precipitation
            pressure
  
    Если параметр не указан, возвращаются все доступные данные.
